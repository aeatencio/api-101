# MariaDB + Adminer

```bash
git clone https://github.com/aeatencio/mariadb-101
cd mariadb
docker-compose up
```

Wait for it to initialize and visit <http://localhost:8080>

Use these credentials:

- Username: `root`
- Password: `example`

Read more in [MariaDB Docker Official images](https://hub.docker.com/_/mariadb)
